# Roadmap

## Phase 1: Commercial Foundation

Deliver app skeleton, tenant-aware architecture, Supabase schema/RLS, app shell, public tenant landing, Admin Master overview, and owner dashboard with realistic data contracts.

## Phase 2: Scheduling Core

Deliver service catalog, calendar availability engine, public booking, internal calendar views, appointment creation/edit/cancel, and status history.

## Phase 3: Operations

Deliver clients, pets, employees, veterinarians, checklist workflow, tutor tracking token, and realtime timeline.

## Phase 4: Billing and Governance

Deliver manual Pix subscription workflow, Admin Master payment confirmation, tenant blocking/reactivation, audit trails, and support access logging.

## Phase 5: Production Hardening

Deliver storage rules, automated tests, seed data, deployment configuration, monitoring, browser regression checks, and security review.
